# Registro Gym

Miniapp web local-first para registrar rutinas de gimnasio desde el móvil.

## Publicación

Es una app estática: se puede publicar tal cual en GitHub Pages, Netlify, Vercel o cualquier hosting de archivos estáticos.

Archivos necesarios:

- `index.html`
- `styles.css`
- `app.js`
- `manifest.webmanifest`

## Datos

Los registros se guardan en el navegador del dispositivo. Usa `Ajustes > Exportar copia` para guardar un backup JSON, y `Exportar CSV` para llevar sesiones a Google Sheets.
